Lend-A-Hand

This is an Android application designed to connect people in need with volunteers willing to help within their local community. The app allows users to create help requests, browse available tasks, and offer assistance, encouraging collaboration and social responsibility through technology. The application features user authentication, structured request posting, and real time data updates, ensuring a smooth and responsive user experince. Built with a focus on usability and scalability, Lend-A-Hand demonstrates practical mobile development skills and the ability to design solutions that address real world problems.

Snake AI Assignment

This project implements an autonomous Snake game agent that navigates a grid-based environment to collect food while avoiding collisions with walls, other snakes and its own body. The AI models the game board as a graph and uses the A* algorithm to compute the shortest safe path to the food at each step.
