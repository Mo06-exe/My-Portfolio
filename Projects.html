Lend-A-Hand

An Android application designed to connect people in need with volunteers within their local community. Users can create help requests, browse available tasks, and offer assistance.

Features include user authentication, structured request posting, and real-time data updates. Built with usability and scalability in mind, this project demonstrates practical mobile development and problem-solving skills.

Snake AI Assignment

An autonomous Snake game agent that navigates a grid-based environment to collect food while avoiding collisions.

The AI models the board as a graph and uses the A* algorithm to compute the shortest safe path at each step.
